## ✨ Features

* **Responsive Design:** Fully optimized for all screen sizes, including desktops, tablets, and smartphones.
* **Interactive UI:** Smooth transitions, hover effects, and dynamic components powered by JavaScript.
* **Comprehensive Sections:**
    * **Hero Section:** A welcoming banner with a clear Call-to-Action (CTA) for booking appointments.
    * **Services:** Detailed grid showcasing medical specialties (e.g., Cardiology, Pediatrics, Neurology).
    * **About Us:** Information regarding the hospital's mission, vision, and experienced medical staff.
    * **Doctors/Team:** Profiles of specialized doctors with their designations.
    * **Contact & Booking:** An interactive contact form for user inquiries and appointment scheduling.

---

## 🛠️ Technologies Used

This project was built using pure frontend technologies with no external heavy frameworks, ensuring fast loading times and clean code.

* **HTML5:** Structured the semantic layout of the web pages.
* **CSS3:** Styled the user interface using custom layouts, Flexbox/Grid, and responsive media queries.
* **JavaScript (ES6+):** Handled dynamic behavior, form validation, and interactive UI components.

## ⚙️ How to Run Locally

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/healthcare-website.git](https://github.com/your-username/healthcare-website.git)
    ```
2.  **Navigate into the project directory:**
    ```bash
    cd healthcare-website
    ```
3.  **Open the project:**
    Simply double-click the `index.html` file to view it in your browser, or open the folder in an editor like VS Code and use the **Live Server** extension.
